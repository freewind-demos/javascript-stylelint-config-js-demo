JavaScript Stylelint "stylelint.config.js" Demo
===============================================

在Stylelint中使用配置文件`stylelint.config.js`。

使用`.js`格式的配置文件，可以让我们在配置时使用JS语法从而获得更大的灵活性。

```
npm install
npm run demo
```
